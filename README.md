# 📊 Sales Performance Analysis (Excel)

## 📌 Overview

This project analyzes sales performance using Excel. It includes data cleaning, data modeling, pivot tables, and an interactive dashboard.
---

## 📁 Dataset

The dataset includes:

* Customers
* Products
* Orders
---

## 🛠 Tools Used

* Excel
* Pivot Table
* XLOOKUP
* Dashboard & Slicers
---

## 📊 Key Analysis

* Revenue trend over time
* Top-performing products
* Profit by category
* Sales performance by salesperson
---

## 📈 Dashboard Features

* KPI: Total Revenue, Profit, Orders
* Interactive filters (Segment, Category, City)
* Charts: Line, Bar, Pie
---

## 💡 Insights

* Laptop category generates the highest profit
* Dell XPS 13 is the top-performing product
* Corporate segment contributes the most revenue
---

## 🚀 Skills Demonstrated

* Data cleaning
* Data modeling (star schema)
* Data analysis
* Data visualization
---
